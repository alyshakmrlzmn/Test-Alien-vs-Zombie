## User Manual

SETTING UP THE GAME

Customize your own board , from the type of the game you like to play to how much zombies you want to put in.

COMMAND

up     - Move up.
down   - Move down.
left   - Move left. 
right  - Move right.
arrow  - Change the direction of an arrow.
help   - Display these user commands.
save   - Save the game. 
load   - Load the game.
quit   - Quit the game.
